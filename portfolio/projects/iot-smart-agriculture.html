IoT Smart Agriculture System

A sensor-driven crop monitoring platform tracking soil moisture, temperature, humidity, rainfall, water level, NPK nutrients, and light intensity — with automated irrigation and real-time alerts for farm management that doesn't require standing in the field.

01 Overview

This system moves crop monitoring from manual, walk-the-field checks to continuous, sensor-driven visibility. A Raspberry Pi coordinates a set of environmental sensors, streams readings to a cloud dashboard, and can act on what it sees — turning on irrigation automatically rather than waiting for someone to notice dry soil.

02 The problem

Crop health depends on several environmental variables at once — moisture, nutrients, temperature, rainfall — and by the time a visible problem shows up in the field, the underlying condition has often existed for days. Manual monitoring can't realistically track all of these continuously, and reactive irrigation wastes water and risks crop stress. The system was built to close that gap with constant sensing and automated response.

03 Architecture

01
Sensor layer
Tracks soil moisture, temperature, humidity, rainfall, water level, NPK nutrients, and light intensity via IoT sensors wired to a Raspberry Pi.
02
Edge coordination
The Raspberry Pi reads sensor data locally and pushes it upstream, keeping the field-side logic lightweight and reliable.
03
Cloud dashboard
Sensor data is exposed through a REST API to a cloud dashboard for remote, real-time visibility into field conditions.
04
Automated irrigation & alerts
Irrigation control triggers automatically from live sensor thresholds, with real-time alerts flagging conditions that need attention.

Built in Python running on a Raspberry Pi, with a REST API connecting sensor data to a cloud-hosted dashboard.

04 What it handles

  • Continuous monitoring of soil moisture, temperature, humidity, and rainfall
  • NPK nutrient and light intensity tracking
  • Water level monitoring
  • Automated irrigation control based on live thresholds
  • Real-time alerts for remote farm management
  • Cloud dashboards for optimizing water usage and crop productivity
